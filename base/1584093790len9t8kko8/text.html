Если вы никогда раньше не использовали git, сначала нужно выполнить некоторые настройки. Выполните следующие команды, чтобы git знал ваше имя и адрес электронной почты

$ git config --global user.name "Your Name"

$ git config --global user.email "your_email@whatever.com"

Настройки конца строки

Для Windows:

$ git config --global core.autocrlf true

$ git config --global core.safecrlf true

Для Unix

git config --global core.autocrlf input

git config --global core.safecrlf true
